# tpc_c_cplusplus

## 简介
本仓库主要用于存放已经适配OpenHarmony的C/C++三方库的适配脚本，三方库的适配/使用指导；每个三方库目录下包含了GN化的适配指导、编译脚本和包含了cmake的适配指导、编译脚本。



## 本仓库目录

```
tpc_c_cplusplus
├── README_zh.md
├── common                #通用说明文档的文件夹
├── 三方库1
│   ├──gn                 #gn构建脚本、适配指导的文件夹
│   ├──cmake              #cmake构建脚本、适配指导的文件夹
│   ├──README_zh.md       
│   ├──README.OpenSource  #文件中包含了三方库源码的下载地址，版本，license等信息
├── 三方库2
│   ├──gn                 #gn构建脚本、适配指导的文件夹
│   ├──cmake              #cmake构建脚本、适配指导的文件夹
│   ├──README_zh.md       
│   ├──README.OpenSource  #文件中包含了三方库源码的下载地址，版本，license等信息
......
```
