# xerces-c三方库说明
## 功能简介
xerces是一个开放源代码的XML语法分析器，它提供了SAX和DOM API。
## 使用约束
- ROM版本：OpenHarmony-v3.2-Beta1
- IDE版本：DevEco Studio 3.0 Release
- API版本：9
- 三方库版本：v3.2.3
- 当前适配的功能：支持sax解析和dom解析
- [Apache License 2.0](https://github.com/apache/xerces-c/blob/master/LICENSE)
