# minizip-ng 三方库说明
## 功能简介
minizip是一个用C编写的zip文件操作库。
## 使用约束

- IDE版本：DevEco Studio 3.0 Beta4
- ROM版本：OpenHarmony3.2 Beta1
- API版本：9
- 三方库版本：3.0.4
- 当前适配的功能：完成了创建和解压BZIP2、LZMA、XZ和ZSTD格式的zip文件
- [License : zlib](https://github.com/zlib-ng/minizip-ng/tree/3.0.4/LICENSE)
